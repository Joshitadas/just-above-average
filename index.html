AI Agents Explained

1. What are AI Agents?

An AI Agent is a digital worker that can understand natural language instructions and autonomously take actions to complete tasks. Think of them as intelligent bots that can mimic how a human might approach a task.

They can perform tasks such as:

  • Searching through a database
  • Updating spreadsheets
  • Sending emails
  • Generating documents
  • Booking appointments

2. What is an Agent Made Of?

An AI agent typically consists of the following components:

  • LLM (Large Language Model) ‚Äì Acts as the brain of the agent
  • Prompt ‚Äì Natural language instructions that guide the agent's behavior
  • Memory ‚Äì Stores short-term or long-term context
  • External Knowledge ‚Äì Domain-specific information like APIs or documents
  • Tools ‚Äì Functions or APIs that let the agent take real-world actions

3. What Do We Focus On?

With powerful LLMs like GPT-4 taking care of the "brain" and "memory", most of our focus shifts to:

  • Prompts ‚Äì Crafting clear and structured instructions
  • External Knowledge ‚Äì Providing the agent with domain-specific data and context
  • Tools ‚Äì Giving agents the right APIs or functions to perform real tasks

4. What Are Tools?

Tools are how agents interact with the outside world ‚Äî they are APIs or functions that perform specific actions. Each tool typically comes with a schema, which tells the agent how to use it.

5. How Does the Agent Work?

AI agents generally follow a Plan ‚Üí Act ‚Üí Reflect ‚Üí Plan loop (PARP loop). Here's a breakdown:

üß™ Example Prompt:
"Send a meeting invite to Alice for Friday at 3 PM."
  1. Plan: Decide it needs to check the calendar first.
  2. Act: Calls check_calendar() and then send_email().
  3. Reflect: Confirms if the email was sent and calendar slot is booked.
  4. Plan Again: If any step fails, re-plan or escalate.
